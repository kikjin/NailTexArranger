# NailTexArranger

[English readme](readme.md)

NailTexArranger（ネイルテックスアレンジャー）は、ネイルテクスチャを3Dアバターに適用することで、3Dアバターの変更プロセスを簡略化するために設計された一連のPythonスクリプトです。これは、各指の爪のテクスチャをマスク画像を使用してクリッピングし、別のアバターのテクスチャに配置することで実現されます。

## 機能

- 各指の爪のテクスチャをクリッピングします。
- クリッピングしたテクスチャをアバターの本体テクスチャに適用します。
- 3Dアバターの爪の変更プロセスを効率化します。

## 必要条件

- Python 3.x
- OpenCV (cv2)
- NumPy
- PyYAML

## 使用法

1. このリポジトリをローカルマシンにクローンするかダウンロードします。

1. Python 3.xがインストールされていることを確認し、必要なライブラリ（`cv2`、`NumPy`、`PyYAML`）もインストールします。

1. 各指の爪のテクスチャをクリッピングするためのマスク画像を準備します。その際、`generate_masks.py` を使用できます。

1. `generate_masks.py` を使用するには、クリッピングしたい部分に特定の色を塗った画像を作成する必要があります。塗る色は `name_color_mapping.py` で定義されています。

1. `clip_image.py` を実行して、各指の部分に爪のテクスチャをクリッピングします。

1. 各爪のテクスチャを配置する場所を定義するYAMLファイルを準備します。その際、`generate_arrangement_settings.py` を使用できます。

1. `arrange_images.py` を実行して、クリッピングされた爪のテクスチャをアバターの本体テクスチャに配置します。

## 初めてのステップ

### クローンまたはダウンロード

このコマンドを実行して、このリポジトリをクローンするか、単に「Download ZIP」ボタンをクリックします。
```bash
git clone https://github.com/kikjin/NailTexArranger.git
```

### Pythonと必要なライブラリのインストール
OSに合わせてPython 3.xをインストールします。  
OpenCV、NumPy、PyYAMLを次のようにインストールします：
```bash
pip install opencv-python numpy pyyaml
```

### クリッピングと配置のための画像の準備
爪のテクスチャとアバターの本体テクスチャに合わせて2つの画像を準備する必要があります。爪のテクスチャの各部分に対し、クリッピングする領域を指定するために特定の色で塗ります。さらに、アバターテクスチャに貼り付ける場所を指定するために、各指に対応する同じ色を塗って貼り付け先領域を指定します。  
塗る色は `name_color_mapping.yaml` で定義されています。切り取り/貼り付け先領域を認識するための色を変更するには、このファイルを編集して下さい。
切り抜き元は爪の形状に合わせるか、四角形で指定します。四角形で指定した場合、貼り付け先のUV配置によっては隣の爪と干渉する場合があります。  
貼り付け先は四角形領域で指定することをお勧めします。

### マスク画像の生成
次のようにして、クリッピングするためのマスク画像を生成します：
```bash
python generate_masks.py [マスク画像のパス]
```
生成されたマスクは masks_MASK_IMAGE_NAME フォルダに保存されます。

### 爪のテクスチャのクリッピング
作成したマスク画像を使用して、爪のテクスチャをクリッピングします。

```bash
python clip_image.py [ソース画像のパス] [マスクフォルダのパス]
```
クリッピングされた画像は `clipped_SOURCE_IMAGE_NAME` フォルダに保存されます。

### arrangement_settings.yamlの生成
次のようにして、アバターの arrangement_settings.yaml を生成します：

```bash
python generate_arrangement_settings.py [ソース画像のパス]
```
arrangement_settings_SOURCE_IMAGE_NAME.yaml が作成されます。

### クリッピングされた画像の配置
```bash
python arrange_images.py [配置設定ファイル] [クリッピングされた画像のフォルダ] [合成画像（オプション）]
```
透明な画像に配置された爪のテクスチャが作成されます。合成画像（アバターの本体テクスチャ）を指定した場合、配置された爪のテクスチャが含まれたアバターテクスチャも作成されます。
